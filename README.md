# Python API

Prepare environment:

```
sudo apt-get update
sudo apt-get install python3
sudo apt-get install python3-pip
sudo apt-get install python3-venv
```

Install dependencies:

```
pip3 install pylint
python3 -m venv env
pip3 install -r requirements.txt
```

Run it:

```
python3 start.py
```
